== Description ==

Execute SQL commands via simple interface.  It has error handling and reporting as well as some elementray security.


== Installation ==

1. Upload and Active Plugin
2. Go to Settings > Run SQL
3. Enter in SQL and hit sumbit
4. Enjoy!
